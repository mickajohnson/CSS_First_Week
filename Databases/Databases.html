What are Databases?

A structured set of data

Databases are a collection of information, they are used for creating, manipulating, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or non-relational database. Relationships means a set of our data is related to another section of our data. Meanwhile in a non-relational database there is not a common related section of our data. Our databases are put into sections called schemas, inside each schema each data set is organized into tables, those tables contain columns and rows to house that particular set of data.

Storage

All databases are just a bin of storage. This storage is housed on your harddrive, often times large companies will have floors upon floors of just database storage (think NSA styled). Our computer stores information on the harddrive, we are able to create new information, read information, update information, and delete information

Interaction

The amazing part about having a Database is that it is a storage bin that holds onto all saved data. Well if only one user had access to that data its just like what we have built on or own computers, it wouldnt be useful to other users. However the joy of havng a database is that it is a storage bin many users can interact with, creating, reading, updating, and deleting information that all users would see.
